========================================================================
    КОНСОЛЬНОЕ ПРИЛОЖЕНИЕ. Обзор проекта ConvolutionalNNetwork
========================================================================
Проект для исследования архитектуры сверточных нейронных сетей
Позволяет создавать сверточную НС с произвольным количеством и чередованием convolutional/subsampling(max pooling) слоев,
к которой присоединена полносвязная сеть обратного распространения
Для каждого слоя можно задавать скорость обучения, количество нейронов
Обучение сверточной сети осуществляется градиентным методом
